# Digital Life Lessons — Client

## Live URL

Client Live Site: `https://digital-life-lessons-moeen.vercel.app`

## Project Purpose

Digital Life Lessons is a modern full-stack web platform where users can create, save, share, and explore meaningful life lessons. The client application provides the user interface for browsing public wisdom, managing personal lessons, upgrading to Premium, interacting with content, and using both user and admin dashboards.

The goal of this frontend is to deliver a recruiter-friendly, responsive, and visually polished experience with a dark wisdom-ledger theme, protected dashboard pages, public lesson discovery, and smooth interaction with the backend API.

## Key Features

- Responsive dark/light themed user interface
- Home page with hero slider and animated sections
- Dynamic Featured Life Lessons section controlled by admin
- Dynamic Top Contributors of the Week section
- Dynamic Most Saved Lessons section
- Public Lessons page with search, category filter, emotional tone filter, sorting, and pagination
- Premium lesson lock/blur UI for Free users
- Login and registration pages with email/password and Google login support
- Protected dashboard layout for users and admins
- Add Lesson, My Lessons, Update Lesson, and My Favorites pages
- Lesson Details page with author card, total lessons created, metadata, likes, favorites, comments, reports, social sharing, and PDF export
- Pricing / Upgrade page with Free vs Premium comparison
- Payment success and cancel pages
- Admin dashboard pages for analytics, users, lessons, reports, and profile
- Custom 404 Not Found page
- Toast and SweetAlert messages instead of browser alert
- Modern card-based UI with glassmorphism styling
